# Portfolio Frontend

A modern, cyberpunk-themed portfolio website built with React and Tailwind CSS v4.

## Features

✨ **Modern Tech Stack**
- React 19 with Vite 8
- Tailwind CSS v4 with PostCSS
- React Router v7 for navigation
- Animated background particles
- Responsive design

🎮 **Cyberpunk UI/UX**
- Retro terminal aesthetics
- Glowing neon effects
- Pixel-perfect borders
- Custom animations
- Grid background patterns

📱 **Responsive & Accessible**
- Mobile-first design approach
- Semantic HTML structure
- Keyboard navigation support
- ARIA labels and roles

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <your-repo-url>
cd portfolio-project/frontend
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run test` - Run tests with Vitest

## Project Structure

```
src/
├── components/       # Reusable UI components
│   ├── Header.jsx   # Navigation header

│   ├── Layout.jsx   # Page layout wrapper
```

## Styling & Theme

Uses Tailwind CSS v4 with a custom cyberpunk color scheme:
- **Primary**: Cyan (#00f5ff)
- **Background**: Dark blue/black (#0a0a0f)
- **Accents**: Purple and green highlights
- **Typography**: Mix of Inter, Fira Code, and Press Start 2P fonts
